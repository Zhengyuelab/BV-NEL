# Example files

This folder contains only files exported or captured directly from the mobile-side workflow.

| File/folder | Description |
|---|---|
| `records/` | Example TXT file exported from the record-management page. |
| `screenshots/` | Example screenshots showing the result page and record page. |

Intermediate OCP values, coarse/refined i-t currents, and fitting tables are not included as example files in this repository.
