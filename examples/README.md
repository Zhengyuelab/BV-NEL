# Example files

These files are included to illustrate the expected GitHub data layout and exported file formats.

| File/folder | Description |
|---|---|
| `ocp_raw_data_example.csv` | Generated OCP example data: 40 s, 0.1 s interval. |
| `coarse_it_raw_data_example.csv` | Generated coarse i-t example data: 3 i-t steps, 8 s per step, 0.1 s interval. |
| `fine_it_raw_data_example.csv` | Generated fine i-t example data: 5 i-t steps, 8 s per step, 0.1 s interval. |
| `step_average_and_fitting_results.csv` | Averaged current and fitting-output example. |
| `timing_schedule_example.csv` | Strict timing schedule matching the app workflow, including 5 s inter-i-t delay and 1.5 s post-stop delay. |
| `records/` | TXT export example from the V6 record-management page. |
| `screenshots/` | Example screenshots of the result page and record page. |
| `diagnostics/` | Example diagnostic log format. |

The CSV files are generated demonstration data, not final experimental data.
