# 通讯协议 V1.00

## 1、查找仪器
* **发送命令**：`55 aa 00 00 00 00 00 01 01 00 00 00 0D 0A` *(new 20240225)*
* **找到仪器，下位机回复**：`55 aa 00 00 00 00 01 01 01`
    *(以下为 AD 采样系数和零点数据)*
    * `3F 80 00 00` (电压系数，float)
    * `3F 81 47 AE 3F 82 8F 5C 3F 83 D7 0A 3F 85 1E B8 3F 86 66 66 3F 87 AE 14 3F 88 F5 C2 3F 8A 3D 70 3F 82 8F 5C` (电流系数，一共九档，float)
    * `00 00 00 00` (电压零点mV，float)
    * `3F 80 00 00 40 00 00 00 40 40 00 00 40 80 00 00 40 A0 00 00 40 C0 00 00 40 E0 00 00 41 00 00 00 41 00 00 00` (电流零点nA，一共九档，float)
    * `0D 0A`
* **实测兼容说明**：小电流设备可能返回固定 91 字节响应，尾部为 `00 00` 而不是 `0D 0A`；解析时应优先按固定长度处理。

## 2、电流量程切换
* **发送命令**：
```hex
55 aa 00 00 00 00 00 02 02 00 0X 0D 0A
```
* **说明**：`0X` 此处代表电流量程码，分别对应以下量程：

| 电流量程码 | 电流量程 | 计算公式 |
| :--- | :--- | :--- |
| 02 | 10mA | Data / 100 |
| 03 | 1mA | Data / 1000 |
| 04 | 100uA | Data / 10000 |
| 05 | 10uA | Data / 100000 |
| 06 | 1uA | Data / 1000000 |
| 07 | 100nA | Data / 10000000 |
| 08 | 10nA | Data / 100000000 |

## 3、开始采样
* **发送命令**：
```hex
55 aa 00 00 00 00 00 03 08 01 00 0D 0A
```

## 4、停止采样
* **发送命令**：
```hex
55 aa 00 00 00 00 00 03 08 02 00 0D 0A
```
* **注意**：默认静置时间 1s 起步，设置 0 秒是内部设置 1s。

## 6、返回数据格式

**例**：`55 AA 00 00 00 00 2A CF BD 21`

* **2ACF** 是十六进制（最高位是符号位，0代表正值，1代表负值），转换成10进制是 `10959`。
    * 计算成电压是：`10 * 10959 / 32768 = 3.344`
    * 说明：其中 `10` 是电压（电流）满度量程，根据参数设置中的电压或者电流量程来选择。`32768` 是 AD 分辨率。
* **BD21** 是十六进制（最高位是符号位，0代表正值，1代表负值），这里需要取反为 `-42DE`，转换成10进制是 `-17105`。
    * 计算成电压是：`10 * (-17105) / 32768 = 5.22`。

## 7、根据方法解析数据包

1.  **LSV 和 CV 试验时**：
    * 当“扫描速率*1000/扫描间隔”（以下称为信号频率）<= 1000 时，一个数据包中包含 20 组数据，格式如下：
        `55 AA 00 00 00 00 [2A CF BD 21]x20` （此时数据取后面 10 个数据做平均显示）。
    * 当“信号频率” > 1000 时，一个数据包格式不变，只有一组数据：
        `55 AA 00 00 00 00 2A CF BD 21`。
2.  **SCV 试验时**：
    * 不管信号频率多少，一个数据包中包含 20 组数据，格式如下：
        `55 AA 00 00 00 00 [2A CF BD 21]x20` （此时数据取后面 10 个数据做平均显示）。
3.  **在开路方法中**：
    * 当“1/扫描间隔”（以下称为信号频率）<= 1000 时，一个数据包中包含 20 组数据（此时 20 个数据做平均显示）。
    * 当“信号频率” > 1000 时，一个数据包格式不变，只有一组数据：
        `55 AA 00 00 00 00 2A CF BD 21`。
    * 当“信号频率” > 10000 时，一个数据包中包含 5 组数据（此时这里面表示 5 个数据依次显示，不需要处理，只是为了高速传输效率）。
4.  **I-T 曲线实验时**：
    * 采样间隔 t >= 0.002s 时，默认采样频率 500Hz。
    * 采样间隔 t < 0.002s 时，采样频率 1/t，软件进行滤波处理。
5.  **其他实验时（不包含 NPV、DPV 和 SWV）**：
    * 当参数中“采样间隔” >= 0.0001S 时，只有一组数据：
        `55 AA 00 00 00 00 2A CF BD 21`。
    * 当参数中“采样间隔” < 0.0001S 时，一个数据包中包含 5 组数据（此时这里面表示 5 个数据依次显示，不需要处理，只是为了高速传输效率）。

## 12、设置设备名称
* **发送命令**：
```hex
55 aa 00 00 00 00 00 09 01 00 05 0D 0A
```
* **名称映射**：
    * `00 05` 代表名称为 SIGNAL NB-A05
    * `01 05` 代表名称为 SIGNAL NB-A15
    * `0A 05` 代表名称为 SIGNAL NB-A105
    * `01 12` 代表名称为 SIGNAL NB-A118
    * `12 14` 代表名称为 SIGNAL NB-A1820
* **说明**：最多容纳 255254 台设备，0d 0a 不符合。
