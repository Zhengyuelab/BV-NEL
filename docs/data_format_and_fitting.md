# Example output files

The `examples/` folder is limited to files that are directly exported or captured from the mobile-side workflow.

## Record export example

`examples/records/V6-BV_NEL_records_20260707_234918.txt`

This TXT file demonstrates the record-export format from the V6 record-management page. It uses tab-delimited columns: No., sample name, time, Eh, and fit R².

## Screenshots

`examples/screenshots/`

This folder contains example screenshots of the result page and record-management page. These screenshots are provided only to illustrate the interface and exported-result display.

## Intermediate data

Intermediate OCP values, coarse/refined i-t currents, and fitting tables are not included as example files in this repository. The measurement workflow and fitting logic are documented in `docs/measurement_workflow.md`, `docs/parameter_table.md`, and the source code.
